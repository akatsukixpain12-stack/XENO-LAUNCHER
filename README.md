# Xeno Launcher

Xeno Launcher is a high-performance, premium-feel Minecraft launcher featuring a strictly cracked mode, universal version support, and integrated mod management.

## 🚀 Features

- **Strictly Cracked & Universal**: Play any version of Minecraft (Alpha, Beta, Release, Forge, Fabric) without a premium account. Bypasses "Demo" UI locks automatically using universal offline tokens.
- **Xeno Mod Center**: Browse and install mods directly from CurseForge and Modrinth within the launcher.
- **Universal Auth**: Supports Microsoft, Mojang, and Ely.by accounts.
- **Premium Experience**: Bouncy UI interactions, custom branding, and optimized JVM arguments for better FPS performance.
- **Customizable**: Easy renaming of offline accounts directly from the landing screen.

## 🛠️ Developer Setup

If you want to build the launcher from source:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/<your-username>/XenoLauncher.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run in development mode**:
   ```bash
   npm start
   ```
4. **Build the installer**:
   ```bash
   npm run dist
   ```
