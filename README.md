## Features

- **Ely.by Authentication** — Sign in with your Ely.by account to play on cracked servers
- **Offline Mode** — Play without any account, just enter a username
- **Microsoft Auth** — Full Microsoft (OAuth 2.0) authentication for premium accounts
- **Every Version** — Play any Minecraft version from Alpha to the latest release and snapshots
- **Lunar Client-Inspired UI** — Dark theme, sidebar navigation, version cards, and modern design
- **Memory Management** — Allocate RAM with an intuitive slider
- **Game Resolution** — Set custom resolution and fullscreen mode
- **Auto Java** — Automatic Java detection and configuration
- **Auto Updates** — The launcher keeps itself up to date

## Installation

### Requirements

- [Node.js v22](https://nodejs.org/)
- [Git](https://git-scm.com/)
- npm (comes with Node.js)

### Setup

```console
git clone https://github.com/akatsukixpain12-stack/XENO-LAUNCHER.git
cd XENO-LAUNCHER
npm install
```

### Run

```console
npm start
```

### Build Installers

```console
# Windows
npm run dist:win

# macOS
npm run dist:mac

# Linux
npm run dist:linux
```

## Authentication Methods

### Ely.by (Cracked)
1. Create a free account at [ely.by](https://ely.by/register)
2. Sign in with your Ely.by credentials in the launcher
3. Play on any cracked server

### Offline Mode
1. Select "Offline" tab in the login screen
2. Enter any username (3-16 characters)
3. Play singleplayer or on offline-mode servers

### Microsoft (Premium)
1. Select "Microsoft" tab
2. Sign in with your Microsoft account
3. Requires a purchased copy of Minecraft
